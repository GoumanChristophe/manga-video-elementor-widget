# Manga Video Widget pour Elementor

## À propos du Projet
Ce projet fournit un widget personnalisé pour Elementor, appelé Manga Video Widget. Il permet aux utilisateurs d'Elementor d'intégrer facilement des vidéos de manga et des boutons de navigation associés dans leurs pages WordPress. Le widget offre diverses options de personnalisation, telles que le style des boutons et la gestion de l'affichage des vidéos.

## Construit Avec
 - PHP
 - WordPress
 - Elementor
## Commencer
Pour utiliser ce widget dans votre site WordPress, suivez les étapes ci-dessous.

### Prérequis
 - Une installation fonctionnelle de WordPress
 - Le plugin Elementor installé sur votre site WordPress
   
### Installation
1. Téléchargez le fichier class-manga-video-widget.php.

2. Placez le fichier dans le répertoire de votre thème WordPress, généralement sous wp-content/themes/your-theme/.

3. Incluez le fichier dans le fichier functions.php de votre thème en ajoutant la ligne suivante :

"include_once 'class-manga-video-widget.php';" 

## Utilisation
Après l'installation, le widget Manga Video sera disponible dans l'éditeur Elementor sous la catégorie "General". Vous pouvez glisser et déposer ce widget dans vos pages et configurer les options selon vos préférences.
